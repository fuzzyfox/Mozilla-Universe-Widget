Mozilla Universe Widget

 

Overview

This is a small widget, created with jQuery due to nessesity, that allows you to embed the Mozilla universe map on any webpage.

Note that this is experimental and may not be used for anything official as of yet!

Demonstrations

Currently there are 3 ways to use the widget. Below we will see what these are, as well as the basic implimentation needed to get each option

Common requirement

For any of these options to display you must have the following 2 lines of code in your 

<head>

NOTE: you can only have one instance of the widget per page. You can however have multiple triggers.


<link href="http://github.com/fuzzyfox/Mozilla-Universe-Widget/raw/master/assets/css/mozilla-universe.css" rel="stylesheet" type="text/css">
<script src="http://github.com/fuzzyfox/Mozilla-Universe-Widget/raw/master/assets/js/mozilla-universe.js" type="text/javascript"></script>

Default Theme overlay/lightbox effect

This is the most basic and likely the most common usage of the widget.

To get this you simply have to load the widget and add the 
rel="mozilla-universe"
to any element you want to toggle the widget when clicked. 

<!-- javascript to load the widget -->
<script type="text/javascript">
	mozillaUniverse({});
</script>

<!-- widget trigger / toggle switch -->
<a href="http://www.mozilla.org/community" rel="mozilla-universe">click me to toggle the widget</a>

Tab Theme dropdown effect

This is one of the more styled versions of the widget. It displays a small tab at the very top of the page which, when clicked, opens the widget

To get this you have to set the 
theme
option in the widget like so. 

<!-- javascript to load the widget -->
<script type="text/javascript">
	mozillaUniverse({
		widget : {
			theme : 'tab'
		}
	});
</script>

<!-- widget trigger / toggle switch [optional] -->
<a href="http://www.mozilla.org/community" rel="mozilla-universe">yo can also click me to toggle the widget</a>

Embeded Theme no effects

This option is perfect for your site map page. It simply displays the widget on the page, no on/off.

This widget requires a slightly different approach to the previous options, but not by much. 

<!-- widget container -->
<div id="mozilla-universe-embed">
	<!-- you can place an optional error message here like we have -->
	<p>Oops! It looks like the map you are meant to see wont load... never fear!</p>
	<p><a href="http://www.mozilla.org/community">There is another way to see it here!</a></p>
</div>

<!-- javascript to load the widget -->
<script type="text/javascript">
	mozillaUniverse({
		widget : {
			theme : 'embed'
		}
	});
</script>
